A Y2K-styled web app to track your bodies and find matches with other users.

## Setup

1. **Extract the zip** and open the folder in your terminal

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the app:**
   ```bash
   npm run dev
   ```

4. **Open in browser:** Go to `http://localhost:5173`

## Features

- **Email verification** - Real email verification via Firebase Auth
- Sign up with email, password + name
- Add bodies to your list (first + last name)
- Find who has been with the same bodies as you
- **Sound effects** - Custom sound on match, fart sound on no match ðŸ’¨
- Real-time sync with Firebase (multi-user!)
